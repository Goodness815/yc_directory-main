## <a name="features">ğŸ”‹ Features</a>

ğŸ‘‰ **Live Content API**: Displays the latest blog ideas dynamically on the homepage using dummyjson.com API.

ğŸ‘‰ **Pitch Submission**: Users can submit blog ideas, including title, description, category, and multimedia links (
image or video).

ğŸ‘‰ **View Pitches**: Browse through submitted ideas with filtering options.

ğŸ‘‰ **Pitch Details Page**: Click on any pitch to view its details, with multimedia and description displayed.

ğŸ‘‰ **Editor Picks**: Admins can highlight top startup ideas using the "Editor Picks" feature managed via Sanity Studio.

ğŸ‘‰ **Views Counter**: Tracks the number of views for each pitch instead of an upvote system.

ğŸ‘‰ **Search**: Search functionality to load and view pitches efficiently.

ğŸ‘‰ **Minimalistic Design**: Fresh and simple UI with only the essential pages for ease of use and a clean aesthetic.

and many more, including the latest **React 19** and **Next.js 15** features alongside code architecture and
reusability

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone https://github.com/Goodness815/yc_directory-main
cd yc_directory
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Running the Project**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.
